# Coo Backend
这是基于`React`、`React-Router`、`Redux`、`ImmutableJs`实现的React 管理台系统。

组件采用我个人的组件库 CooReact： <a href="https://github.com/coocssweb/cooReact">https://github.com/coocssweb/cooReact</a>
